YOUTUBE CLIP DOWNLOADER - SIMPLE VERSION
=========================================

HOW TO USE:
-----------
1. Double-click 'download.bat' and paste your YouTube URL
   OR
2. Run: yt-dlp.exe "YOUR_YOUTUBE_URL"
   OR
3. Run: python downloader.py

DOWNLOADED VIDEOS:
------------------
All videos are saved in the 'downloads' folder

UPDATE YT-DLP:
--------------
Download latest version from:
https://github.com/yt-dlp/yt-dlp/releases/latest/download/yt-dlp.exe

Replace the existing yt-dlp.exe file.

EXAMPLE URLS:
-------------
• Regular video: https://www.youtube.com/watch?v=VIDEO_ID
• YouTube clip: https://youtube.com/clip/CLIP_ID
• YouTube Short: https://youtube.com/shorts/VIDEO_ID

For more options, run: yt-dlp.exe --help
